# Face Recognition Models

This directory should contain the face recognition models required by face-api.js.

## Required Files

Download the following files from [face-api.js weights repository](https://github.com/justadudewhohacks/face-api.js/weights/):

1. `face_landmark_68_model-weights_manifest.json`
2. `face_landmark_68_model-shard1`
3. `face_recognition_model-weights_manifest.json`
4. `face_recognition_model-shard1`
5. `tiny_face_detector_model-weights_manifest.json`
6. `tiny_face_detector_model-shard1`

Place all these files in this directory for the face recognition to work properly.
