Help & Documentation

Getting Started

1. Start Recognition

Click the "Start Recognition" button on the Dashboard to begin face detection. Make sure your camera permissions are enabled.

2. Face Detection

The system will automatically detect faces in the camera feed every 3 seconds. Hold still and face the camera for best results.

3. Register New People

When an unknown face is detected, click "Register Now" to add the person to the database with their details.

How It Works

FaceWhiz uses advanced face recognition technology powered by face-api.js:

  • Face Detection: Detects faces in real-time from your webcam
  • Face Descriptors: Creates unique 128-dimensional vectors for each face
  • Matching: Compares detected faces against stored descriptors in the database
  • Recognition: Identifies known people or flags unknown faces for registration

Features

People Management

View all registered people, their details, and manage the database from the People page.

Settings

Adjust recognition threshold and view system status in Settings.

Real-time Recognition

Continuous face detection and recognition from your webcam feed.

Database Storage

Frequently Asked Questions

Why isn't my face being recognized?

Make sure you're registered in the system. Ensure good lighting, face the camera directly, and adjust the recognition threshold in Settings if needed.

Can I delete registered people?

Yes! Go to the People page and click the "Delete" button on any person's card.

What's the recognition threshold?

The threshold determines how strict the matching is. Lower values (0.3-0.5) are more strict, higher values (0.7-0.9) are more lenient. Default is 0.6.

My camera isn't working

Check your browser's camera permissions. Make sure no other application is using the camera, and try refreshing the page.

How accurate is the recognition?

Accuracy depends on lighting conditions, angle, and image quality. For best results, use good lighting and face the camera directly.

Troubleshooting

Camera Issues

  • Grant camera permissions in your browser settings
  • Check if another application is using the camera
  • Try a different browser (Chrome, Firefox, Edge)

Recognition Issues

  • Ensure face-api.js models are loaded (check browser console)
  • Verify backend is running on port 3000
  • Check database connection in Settings page
  • Adjust recognition threshold if getting false matches

Backend Connection

  • Make sure backend server is running: npm run dev
  • Check backend status in Settings page
  • Verify API endpoints are accessible

Support

For additional support or to report issues, please check the browser console for detailed error messages. The AI chatbot on the Dashboard can also help with basic questions.
